취업하고 싶은 사람의
포트폴리오 정리 페이지

의미보다 분위기, 생산성보다 압도감, 기능보다 연출에 모든 걸 몰빵한 데모 페이지입니다. 별 기능은 없는데도 광원, 유리질감, 패럴랙스, 궤도 애니메이션, 마우스 반응형 글로우만으로 '이거 뭐지?' 하고 한 번 더 보게 만드는 감성에 집중했습니다.

80% Unity
75% C#, C++ Syntax
65% Algorithm
Glass Depth 겹겹이 쌓인 블러와 반사광으로 유리 패널 느낌을 강조합니다.
Reactive Glow 마우스를 따라 움직이는 광원으로 공간감이 살아납니다.
Orbit Motion 정적인 페이지인데도 뭔가 계속 돌아가고 있는 듯 보입니다.
Hero Core 중앙 덩어리 하나만 세게 잡아도 페이지 인상이 확 달라집니다.
CSharp Unity Engine Cpp
Game Development
